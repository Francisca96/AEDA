AEDA-Casino

No âmbito da unidade curricular de Algoritmos e Estruturas de Dados foi-nos proposto modelar um problema recorrendo ao paradigma da orientação por objetos e usar a linguagem C++ para implementar a solução correspondente.

Construímos assim uma solução modular para a implementação de um casino, que na versão corrente apenas contém o jogo blackjack, mas foi pensado e projetado para ser possível incorporar outros jogos.
O nosso casino, suporta tanto o modo de jogo em que o jogador interage com bots, como o modo de simulação, em que os bots jogam sozinhos contra o dealer e é possível simular de forma rápida um número elevado de rondas. Os bots baseiam-se em algoritmos de jogo reais, publicados na internet por jogadores profissionais.
Os algoritmos de jogo foram construídos de forma a ser fácil ajustar valores e tentar encontrar um algoritmo de jogo "ideal".
No entanto, nenhum algoritmo revelou dar lucro ao jogador, quando aplicado a longo prazo. O que demonstra que: Ou os algoritmos publicados na internet não revelam todos os "segredos" ou na verdade é impossível ganhar no blackjack.
